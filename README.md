A simple blog creation platform built with **Node.js**, **Express**, and **EJS**, allowing users to create blog posts with images and view them dynamically using server-side rendering.

---

## Features

- Create blog posts with title, description, image, and content
- Image upload handling using Multer
- Dynamic blog listing and individual blog pages
- Server-side rendering with EJS
- File-based persistence using Node.js filesystem
- Bootstrap-based responsive UI

---

## Tech Stack

- Node.js
- Express
- EJS
- Multer
- Bootstrap
- File System (`fs`)
- Git & GitHub

---

## Project Highlights

- Handles multipart form data (files + text)
- Demonstrates asynchronous file operations
- Clean separation between routes, views, and public assets
- Built without a database to focus on core backend concepts

---

## Running the Project Locally

1. Install dependencies
2. Start the server
3. Open the application in the browser

The project runs locally and stores data using the file system.
